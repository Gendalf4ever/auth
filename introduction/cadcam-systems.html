CAD/CAM система

CAD

Computer Aided Design

Компьютерное проектирование для создания индивидуальных проектов, соответствующих анатомическому виду зубов пациента

CAM

Computer Aided Manufacturing

Компьютерное моделирование для создания физических изделий на основе цифровых чертежей

CAD/CAM лаборатория

Высокотехнологичное оборудование для проектирования и создания стоматологических реставраций: коронки, мосты, виниры, хирургические шаблоны для имплантаций и других конструкций.
